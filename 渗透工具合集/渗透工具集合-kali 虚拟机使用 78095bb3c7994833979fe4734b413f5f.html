三、KALI常见服务

1.SSH

1.1 查看kali中的ssh服务是否启动

systemctl status ssh

1.2 如何 启动|关闭|重新启动 ssh 服务

systemctl start |stop|restart ssh

1.3 测试SSH是否连接成功

可以通过连接自己的方式进行确认：ssh kali@127.0.0.1

1.4 开启开机自启动命令

sudo systemctl enable ssh
